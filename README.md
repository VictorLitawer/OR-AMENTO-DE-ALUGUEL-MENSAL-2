# Orçamento de Aluguel Mensal

Este projeto consiste em uma aplicação desenvolvida em Python com o objetivo de realizar o cálculo do orçamento mensal de aluguel, considerando despesas fixas como valor do aluguel, água, luz e internet.

A aplicação foi estruturada com base em princípios de orientação a objetos, utilizando uma classe responsável por armazenar os dados e executar o cálculo do valor total das despesas, aplicando de forma prática o pensamento algorítmico e a organização lógica do código.

O sistema solicita a entrada dos valores pelo usuário, realiza o processamento interno por meio de métodos definidos na classe principal e exibe o resultado final do orçamento mensal, demonstrando de forma clara a relação entre entrada, processamento e saída de dados.

Este projeto foi desenvolvido como atividade acadêmica, com foco no aprendizado da linguagem Python, na aplicação de conceitos fundamentais de programação e na organização adequada de um projeto de software.
